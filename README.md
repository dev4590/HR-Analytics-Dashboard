# HR Analytics Dashboard (Power BI)

## Overview
This project analyzes employee data to understand **attrition** and workforce trends.  
An interactive dashboard is created using **:contentReference[oaicite:0]{index=0}** to help HR teams make better decisions.

---

## Objective
- Analyze employee attrition  
- Identify key factors like age, salary, department, and job role  
- Compare attrition across HR, R&D, and Sales departments  

---

## Dataset
- File: `HR_Analytics.csv`
- Total Employees: 1470
- Includes age, department, salary, experience, and attrition data

---

## Tools Used
- Microsoft Power BI  
- Power Query  
- DAX  
- CSV Dataset  

---

## Key Insights
- Overall attrition rate: **16.1%**
- Sales department has the highest attrition
- Higher attrition in age group **26–35**
- Lower salary employees leave more
- Attrition is highest in the first **0–2 years**
